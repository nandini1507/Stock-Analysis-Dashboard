# Stock-Analysis-Dashboard
A simple and interactive Streamlit dashboard that analyzes stock market trends using yfinance and plotly. It allows users to view candlestick charts, calculate indicators like SMA, EMA, and RSI, check performance summaries, and download the data as CSV. Perfect for beginners learning data visualization and financial analytics.
